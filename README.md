# Resume Parser AI

This project is a GUI-based Resume Parser built using Python and NLP techniques.
It analyzes resume text (manual input or PDF) to extract technical skills.
The system assigns a score based on detected skills and classifies resumes into relevant categories.
Users can view results instantly and export them as a PDF report.
The project demonstrates practical use of NLTK, Tkinter, and PDF processing libraries.
